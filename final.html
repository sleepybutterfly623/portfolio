My Projects!

Here are some of the projects that I completed as a student and my learnings as I explored UI/UX.











Development

Explored React in order to create a list-based interface that enables the user to add items to cart and filter and sort based on provided categories such as price, popularity, type of products, and dietary restrictions.

Personas

Observed and interviewed real users interacting with an interface, created personas based on these users, and illustrated a storyboard to properly understand the concept of persona and how valuable they can be for the design process.







Iterative Design

Worked in a group to design an interactive interface for the emerging startup, Koala and understood the iterative design process flow:

1) Sketching

2) High-fidelity prototyping

3) User testing

4) Prototype revision and application of user feedback

Responsive Redesign

Applied the following criterias to redesign a simple website:

1) Usability

2) Learnability

3) Memorability

4) Conceptual Models
